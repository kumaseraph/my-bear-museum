## 🐛 待修復問題

### 1. 星星按鈕完全失效（重要）

**問題：** 點擊任何星星按鈕（☆）都無法變成 ★（收藏）

**根本原因：** HTML onclick 屬性中的 `event` 參數在瀏覽器環境下為 `undefined`

```html
<!-- 問題所在 -->
<button onclick="toggleStar(featuredBear.collectionNo, event)">☆</button>
```

當瀏覽器執行時，`event` 是隱式全域變數，但在某些環境（嚴格模式/DevTools console）為 `null` 或 `undefined`，導致：
```javascript
event.stopPropagation(); // TypeError: Cannot read properties of null/undefined
```

**修復方案：** 將所有 `onclick="toggleStar(..., event)"` 改為使用 `arguments[0]`

```html
<!-- 修復後 -->
<button onclick="toggleStar(featuredBear.collectionNo, arguments[0])">☆</button>
```

**影響範圍：** 7 處需要修改（line 1645, 2469, 2586, 2842, 2891, 3031）

- 移至**左上方**（top-left），在熊熊縮圖**外面**

- 今日主角區塊的熊熊放大時，星星位置需與未放大時一致（可用 fixed position 或絕對定位於縮圖容器）

**樣式：**
- 金屬感 + 立體效果（可用 CSS gradient、box-shadow、text-shadow 達成）
- 未收藏：立體銀灰色立體星星
- 已收藏：立體金色星星 + 發光動畫
